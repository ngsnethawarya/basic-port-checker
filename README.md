# Basic Port Checker

This project checks whether common ports are open on a host.

## What it does
- Accepts a target IP or hostname
- Checks selected ports
- Reports open or closed status

## Common ports
- 22 SSH
- 80 HTTP
- 443 HTTPS
- 3389 RDP

## Why it matters
Port checking helps identify exposed services and supports basic network defense.

## Skills practiced
- Python sockets
- Network fundamentals
- Service identification
- Defensive security thinking
